# KOSPI 턴어라운드 스크리너

DART 전자공시시스템 실시간 연동 - 부채비율 감소 + 영업이익 증가 + 매출성장 종목 분석

## 원클릭 배포

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/poianopks/kospi-turnaround-screener)

## 기능
- DART API 실시간 재무데이터 조회
- 턴어라운드 점수 자동 산출
- 부채비율/영업이익/매출 성장률 비교
- 다크모드 지원
